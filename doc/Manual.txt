■ゲームの概要

　白い箱のようなものを操作してゴール地点を目指す2Dアクションゲームです。

　文字とエフェクト以外はほとんど単色の四角形で画面を構成しているため、
　見た目はシンプルというかレトロというかシュールな感じになっています。
　（グラフィックで描画するモードもあります）

　マップを編集する機能もあるので、自分で作ったマップで遊ぶこともできます。

　全９フロア（ステージ）
　難度は高めです。

　ゲームパッド推奨


■注意

　このソフトは、フリー画像素材・フリー音楽（効果音）素材を使用してます。
　このソフト内で使用されている画像・音楽（効果音）を、このゲームを遊ぶ以外の用途には使用しないで下さい。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。

　DirectX 9.0 以降
　2Dゲームが快適に動作するCPU
　Direct Graphic 対応のビデオカード
　Direct Sound 対応のサウンドカード


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　Hako.exe を実行して下さい。


■遊び方

　●操作方法

　　左右キー         ... 左右に移動
　　下キー           ... 床の際から下に降りる
　　Z                ... ジャンプ／決定
　　X                ... 低速移動／キャンセル
　　A                ... スコープ
　　S                ... クイックリスタート
　　スペースキー     ... 一時停止（ポーズメニュー）
　　エスケープキー   ... 直ちにゲームを終了


　　ゲームパッド対応
　　ボタン設定は SETTING / PAD CONFIG から行ってください。

　　低速移動ボタンを押しながら左右ボタンを押すと、箱はゆっくり移動します。

　　ポーズメニューの RESTART を選択すれば、直ちにフロアをやり直すことができます。
　　クイックリスタートはこれと同じ機能です。

　　スコープボタンを押しながら上下左右ボタンを押すと、視界を調整できます。
　　スコープボタンだけ押して離すとデフォルトの視界に戻ります。


　　▲ゲームパッドについて

　　　パッドはゲームを起動する前にＰＣに認識させて下さい。
　　　ゲーム中に接続しても、たぶん正しく認識されません。

　　　複数のパッドを接続している場合、ゲーム開始以降、最初にボタンを押したパッドが選ばれます。

　　　▲ボタン設定について

　　　　カーソルが指す機能を割り当てたいボタンを押して下さい。

　　　　Z, X キーを押せば、入力を破棄してメニューに戻れます。

　　　　１つのボタンに重複して機能を割り当てることはできません。
　　　　ボタンが足りなくなったらスペースキーを押して割り当てをスキップして下さい。
　　　　機能を割り当てたくない場合もスキップして下さい。


　●操作のコツ

　　マリオのＢダッシュのように１キャラ分の隙間は走り抜けられます。
　　２キャラ分の隙間も走り抜けられます。

　　床の端や１キャラ分の隙間から下に落ちたいとき、下ボタンを押すと落ちやすくなります。

　　各キャラの座標は内部的に 0.001 ドットで制御しています。
　　見た目は同じでも当たるか当たらないかの違いがあるかもしれません。

　　速度の速い敵は、壁にぶつかった時けっこうめり込みます。
　　当たらないだろうと思ったところでも当たるかもしれません。
　　１キャラ分以上めり込むことはないはずなので、壁の向こうにいれば大丈夫。多分。。


　●マップ上のオブジェクト

　　白い箱 ... 主人公です。

　　黄色っぽい箱 ... 壁です。

　　赤く光る箱 ... 即死です。

　　水色に光る箱 ... ゴール地点です。

　　動く青い箱 ... 敵です。触ると即死です。

　　跳ねる赤い箱 ... 敵です。触ると即死です。

　　廻る緑の箱 ... 敵です。触ると即死です。


　●タイムアタック

　　全フロア制覇すると TIME ATTACK で遊べるようになります。

　　全フロア制覇とかやってられんという場合は SETTING / UNLOCK EXTRA MODE を実行して下さい。
　　（全フロア制覇したことにして TIME ATTACK, EDIT MODE を解放します）


　●マップの自作

　　全フロア制覇すると EDIT MODE からマップの自作ができるようになります。

　　全フロア制覇とかやってられんという場合は SETTING / UNLOCK EXTRA MODE を実行して下さい。
　　（全フロア制覇したことにして TIME ATTACK, EDIT MODE を解放します）


■リプレイ

　リプレイを保存＆再生できます。
　リプレイデータは Replay フォルダにファイルとして出力するので、単独で持ち運びできます。
　リプレイデータの再生には、このゲームプログラムと当該マップデータが必要ですのでご注意下さい。

　ミスしたときのリプレイも保存できるようになりました。
　再スタート後、キーボードのＲキーを押すと直前にミスしたリプレイを保存します。

　リプレイの保存を有効にするには SETTING / SAVE REPLAY DATA を ON にして下さい。
　（各フロアクリア後に保存するか確認するようになります）

　●制限

　　１フロアのプレイ時間が１時間を超えるとリプレイを保存できません。
　　この場合、クリアしても保存の確認画面をスキップします。
　　（ここで言うプレイ時間とは、フロアのスタート又は最後にミスしてからの経過時間を指します）

　　作成したリプレイデータは、同じバージョンでのみ再生可能です。


■タイムアタック

　ランクは以下のようになっています。（最高ランクはＳです）

　　Ｓ > Ａ > Ｂ > Ｃ > Ｄ > Ｅ > Ｆ


■編集モード

　EDIT MODE / EDIT FLOORS からゲームを開始すると編集モードになります。
　通常のプレイと同じように始まりますが、ポーズメニューの EDIT FLOOR が使えるようになります。
　EDIT FLOOR を選択することによりマップ編集を開始しますが、
　操作が面倒ならキーボードのＥキーを押しても同じ効果があるのでお使い下さい。
　編集モードのときはエンディングをスキップします。リプレイも保存できません。

　●マップ編集での操作方法

　　マップ編集では主にマウスを使用します。

　　左クリック ... マウスカーソルの位置に選択しているセルを配置
　　右クリック ... マウスカーソルの位置のセルを空にする。
　　シフトキーを押しながら左ドラッグ ... 画面の移動

　　E ... マップ編集終了
　　W ... マウスカーソルの位置をワープ先として記憶

　●編集ウィンドウの説明

　　MENU POS ... ウィンドウの位置を左端または右端に移動
　　SELECT   ... 選択されているセルを表示
　　X        ... 編集終了

　　選択可能なセル

　　　WL ... 壁
　　　DT ... 即死する壁
　　　ST ... スタート地点
　　　GL ... ゴール地点

　　　★スタート地点は必ず１つだけ配置します。

　　　青いセル

　　　　6 ... 青い箱、スタート時の進行方向は右
　　　　3 ... 青い箱、スタート時の進行方向は右下
　　　　2 ... 青い箱、スタート時の進行方向は下
　　　　1 ... 青い箱、スタート時の進行方向は左下
　　　　4 ... 青い箱、スタート時の進行方向は左
　　　　7 ... 青い箱、スタート時の進行方向は左上
　　　　8 ... 青い箱、スタート時の進行方向は上
　　　　9 ... 青い箱、スタート時の進行方向は右上

　　　　★数字と進行方向の対応はテンキー

　　　赤いセル

　　　　L  ... 赤い箱、スタート時の進行方向は左
　　　　LS ... 赤い箱、スタート時の進行方向は左、左右の移動が遅い
　　　　LF ... 赤い箱、スタート時の進行方向は左、左右の移動が速い
　　　　R  ... 赤い箱、スタート時の進行方向は右
　　　　RS ... 赤い箱、スタート時の進行方向は右、左右の移動が遅い
　　　　RF ... 赤い箱、スタート時の進行方向は右、左右の移動が速い

　　　緑のセル

　　　　T6 ... 緑の箱、時計回り、スタート時の位置は回転軸の右
　　　　T3 ... 緑の箱、時計回り、スタート時の位置は回転軸の右下
　　　　T2 ... 緑の箱、時計回り、スタート時の位置は回転軸の下
　　　　T1 ... 緑の箱、時計回り、スタート時の位置は回転軸の左下
　　　　T4 ... 緑の箱、時計回り、スタート時の位置は回転軸の左
　　　　T7 ... 緑の箱、時計回り、スタート時の位置は回転軸の左上
　　　　T8 ... 緑の箱、時計回り、スタート時の位置は回転軸の上
　　　　T9 ... 緑の箱、時計回り、スタート時の位置は回転軸の右上
　　　　H6 ... 緑の箱、反時計回り、スタート時の位置は回転軸の右
　　　　H3 ... 緑の箱、反時計回り、スタート時の位置は回転軸の右下
　　　　H2 ... 緑の箱、反時計回り、スタート時の位置は回転軸の下
　　　　H1 ... 緑の箱、反時計回り、スタート時の位置は回転軸の左下
　　　　H4 ... 緑の箱、反時計回り、スタート時の位置は回転軸の左
　　　　H7 ... 緑の箱、反時計回り、スタート時の位置は回転軸の左上
　　　　H8 ... 緑の箱、反時計回り、スタート時の位置は回転軸の上
　　　　H9 ... 緑の箱、反時計回り、スタート時の位置は回転軸の右上

　　　　★数字とスタート時の位置の対応はテンキー


　●マップのサイズの変更

　　申し訳ありません、未実装です。
　　というわけでマップデータファイルを直接編集していただくようお願いします。
　　（ペイントなどでリサイズして下さい）

　　マップが大きいほどロード時間や処理は長く重くなります。
　　また、敵の数も多くなるほど処理が重くなります。


　●ワープ機能

　　編集中にミスする度にスタート地点へ戻されるのが煩わしいときは、ワープをお使い下さい。
　　マップ編集中に任意のセルにカーソルを合わせ、キーボードのＷキーを押してその位置を記憶、
　　その後プレイ中に同じくＷキーを押すと、最後に記憶した場所へ瞬時に移動します。
　　そのフロアをクリアするかプレイ中止すると位置はクリアされます。


■マップデータファイル

　マップデータは Map フォルダの配下に .bmp 形式のファイルとして置いてあります。
　なのでウィンドウズ付属のペイントなどで直接編集できます。

　ネタバレを防ぐため最初はダミーデータを置いています。
　編集モードが解放されるとオリジナルのマップデータをコピーします。

　マップデータファイルを直接編集する前にゲームを終了して下さい。
